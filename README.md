# workGPT

This repository serves as a learning sandbox for Next.js, Tailwind CSS, and shadcn/ui. It contains a minimal starter project to experiment with these technologies.

## Getting Started

1. Install dependencies after cloning this repo:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

## Notes

This project was created offline, so you may need to ensure all packages are downloaded by running `npm install`.
